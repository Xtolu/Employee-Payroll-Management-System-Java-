A simple console-based Employee Payroll Management System built with Java.

## Features
- Add employees
- View employee records
- Calculate tax and pension
- Generate payslips
- Object-Oriented Design

## Tech Stack
- Java
- OOP
- Collections
- Console Application

## How to Run
1. Clone the repository
2. Open in VS Code / IntelliJ
3. Compile and run `Main.java`
